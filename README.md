## 项目框架

* 资源文件：resources
  * 图片：graphics
    * etc...
  * 音乐：music
    * etc...
  * 音效：sound
    * etc...
* 代码文件：source
  * 配置文件：data

      * etc...

  * 游戏状态：states
    * 主界面：main_menu.py
    * 载入界面：load_screen.py
    * 关卡界面：level.py
  * 常量：constants.py
  * 音乐 / 音效：sound.py
  * 启动：setup.py
  * 工具：tools.py
* 程序入口：main.py
* 记事本：notes.txt
* 项目概述：README.md
